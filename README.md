# כיתת פיננסים — אתר תכנית חינוך פיננסי

אתר שיווקי + סילבוס ומערכי שיעור עבור התכנית **"חינוך פיננסי בעולמות ה-AI"**
של יהונתן השלי, לחטיבות הביניים ולחטיבה העליונה (כיתות ט'–י"א).

## מה כולל האתר (`index.html`)
- **הירו שיווקי** עם נתוני התכנית (30 מפגשים · 60 שעות).
- **רציונל, מטרות וערכים** ופילוח קהל יעד (תלמידים, הורים, מורים, מנהלים).
- **דרכי הוראה** — שיעורים פרונטליים, משחקים, סימולציות, כלים טכנולוגיים ועוזר AI.
- **סילבוס אינטראקטיבי** — 16 יחידות תוכן + פרויקט גמר, עם מערכי שיעור נפתחים וסינון לפי קטגוריה.
- **פרופיל המרצה**, מדדי הצלחה, חבילות הזמנה (קורס מלא / מודול / מפגש בודד) וטופס יצירת קשר.

## הפעלה
קובץ HTML יחיד ועצמאי. פותחים את `index.html` בדפדפן — אין צורך בהתקנה.
עיצוב RTL בעברית, רספונסיבי, בפלטת תכלת·טורקיז·ירוק.
